## 2.0.1 Log back in to your VM

Follow [set up instructions](../setup.md) to log back into your Nectar VM in either VS Code or your terminal. 

In VS Code: 

1. Type `Ctrl` + `Shift` + `P` to open command palette and select `Remote-SSH: Connect to Host` and select the IP address for your VM from the drop-down list
2. Type in your provided password and hit enter

**OR** 

In a terminal application:

1. Type the following into your terminal, using your allocated VM’s IP address:
    ```default
    ssh <username>@<ip_address>
    ```
2. Type in your provided password and hit enter
